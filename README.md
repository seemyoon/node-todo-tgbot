# Node todo TG-bot

## Project setup

To install dependencies, run the following command:

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
## Migrations 

```bash
# generate migrations 
$ name=add_base_entities yarn run migration:generate

# run migrations
$ yarn run migration:run
```
